A grayscale to normal map converter.

# Usage

Simply run

```bash
gtn <file_name> [OPTIONAL: <strength>(default: 20)]
```

# Building

Simply run

```bash
make
```

to build the executable `gtn`.

## Installation

You can also run

```bash
make install
```

to install the executable on your system.

And also

```bash
make uninstall
```

to remove the executable from your system.
