A grayscale to normal map converter.

# Usage

Simply run

```bash
gtn <file_name> ... [-s <strength> (default: 20)] [-d <output_dir> (default: ./)]
```

You can also run

```bash
gtn -h
```

for a help menu.

# Building

Simply run

```bash
make
```

to build the executable `gtn`.

## Installation

You can also run

```bash
make install
```

to install the executable on your system.

And also

```bash
make uninstall
```

to remove the executable from your system.
